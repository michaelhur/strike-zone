# 스트라이크 존

`스트라이크 존`은 미국 프로야구 `메이저리그`의 경기별, 팀별, 선수별, 심판별 스트라이크 존을 확인할 수 있는 대시보드입니다.

`Strike Zone` provides users strike zone of each and every game, team, player and umpire.
